# Weather App

A clean and responsive weather application that provides real-time weather information for cities worldwide. Built with vanilla JavaScript, this app demonstrates API integration, asynchronous programming, and modern CSS design.


## Features

- **Real-time Weather Data**: Get current temperature, humidity, and wind speed for any city
- **Smart Search**: Search by city name with Enter key support
- **Dynamic Weather Icons**: Visual representation of different weather conditions
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Default Location**: Automatically loads weather for Rome on startup
- **Clean UI**: Modern gradient design with intuitive layout

## Technologies Used

- HTML5
- CSS3 (Flexbox, Gradients, Responsive Design)
- Vanilla JavaScript (ES6+)
- OpenWeatherMap API

## APIs Used

- [OpenWeatherMap Current Weather Data](https://openweathermap.org/current) - For real-time weather information
- [OpenWeatherMap Geocoding API](https://openweathermap.org/api/geocoding-api) - For converting city names to coordinates
